Run:

NPM INSTALL

Em seu CMD digite ipconfig. Copie o endereço do IPV4. Ex.: 192.168.0.0

Após isso dentro de src/localServer crie uma const com o seu nome e o seu endereço ipv4 logo em seguida com a porta 4000.

```js
const urlYourName = 'http://192.168.0.0:4000'

export const globalUrl = urlYourName
```

dentro do arquivo na raiz do projeto existe um arquivo chamado json-server.json. na opção "host" coloque o seu numero ipv4 e digite o comando no terminal -----> NPM RUN SERVER. Se tudo funcionar digite NPX EXPO START e conecte seu dispositivo movel ou emulador.
